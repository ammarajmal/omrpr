# OMRPR Analysis Rules

## Claim Boundaries
- All manuscript numbers must come from `docs/claim_boundary.md`.
- Never report same-run LDV waveform validation unless the source file explicitly supports it.
- `torsion_diff_y_mm` is a differential displacement proxy, not a torsion angle.
- `e20_320rpm` must be labeled separately as high-wind unstable motion.

## Plotting
- Every publication figure must be generated by a committed script.
- Every figure script must save:
  - PNG, 300 dpi
  - PDF or SVG where possible
  - CSV of plotted numerical values where possible

## Reproducibility
- Never edit exported result CSV files manually.
- Use scripts under `analysis/`.
- Use `workflow/Snakefile` for final re-runs when stable.
